Como utilizar nosso simulador:

Este simulador é um instrumento interativo que permite aos usuários explorar a matemática e aprender a usar as operações básicas.

Você poderá encontrar imagens ludicas e destintas para representar a quantidade de números que desejar usar.

Você também poderá encontrar simbolos das Operações básicas da matemática.

Eles possibilitarão que quem estiver utilizando o simulador consiga realizar qualquer uma das operações disponiveis.

*OBS: Não pode ser realizadas contas com vírgulas, números negativos, frações, etc...


Esses simbolos são:

  • Adição: símbolo da adição (+) é usado para representar a operação matemática de somar dois ou mais números. Ele indica que devemos juntar os valores envolvidos.
  • Subtração: O símbolo da subtração é o sinal de menos (-). Ele é usado para representar a operação matemática de subtrair um número de outro.
  • Multiplicação: O símbolo da multiplicação é o (x). Ele é utilizado para representar a operação de multiplicar dois ou mais números.
  • Divisão: O símbolo da divisão é (÷). Ele é usado para representar a operação de dividir um número por outro.

Instruções para usar o simulador:

1. Arraste a figura (borboleta ou flor) que deseja usar para o espaço acima delas.

2. Depois, escolha uma das operações básicas disponíveis.

3. Após isso, você verá o resultado da operação escolhida.
